** 터미널에서 웹서버 실행하기
python manage.py runserver

터미널에서 웹서버 종료하기
Ctrl+C

App 생성하기
django-admin startapp testapp

django 프로젝트 생성하기
django-admin startproject config .

버전확인
conda list | findstr django 
django-admin --version

가상환경 목록보기
conda env list

django 가상환경 만들기
conda create --name django

생성한 django 가상환경으로 활성화
conda activate django

가상환경 비활성화
conda deactivate
